# Third Person Mod

A simple Minecraft micro mod to force the user to play in first person mode

Can be used stand alone but highly recommended to be used with a third person
view improvement mod such as: Shoulder Surfing Reloaded
https://minecraft.curseforge.com/projects/shoulder-surfing-reloaded
